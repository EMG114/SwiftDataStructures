  • |>(_:_:)

    The cons operator.

    This operator lazy. i.e:

    func printAndGiveList() -> List<Int> {
  print(2)
  return .Nil
}

2 |> 1 |> printAndGiveList()

    Will not print 2.

    • Complexity: O(1)
