  • LazyList

    A singly-linked, recursive list. Head-tail decomposition can be accomplished with a switch statement:

    extension LazyList {
  public func map<T>(f: Element -> T) -> LazyList<T> {
    switch self {
    case .Nil: return .Nil
    case let .Cons(x, xs): return f(x) |> xs().map(f)
    }
  }
}

    Where |> is the cons operator.

    Operations on the beginning of the list are O(1), whereas other operations are O(n).
